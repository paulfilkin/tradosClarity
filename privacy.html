tradosClarity Privacy Policy

Effective Date: June 27, 2025

Last Updated: June 27, 2025

Privacy Commitment: tradosClarity is designed with privacy by design principles. We do not collect, store, or transmit any personal data or user information to external servers.

What Data We Collect

Personal Data: None. tradosClarity does not collect any personal information, usage analytics, or tracking data.

Local Preferences: The extension stores your keyboard shortcut preferences locally in your browser using Chrome's storage API. This data never leaves your device.

How We Use Your Data

The only data stored is your keyboard shortcut preferences, which are used solely to:

  • Remember your custom keyboard shortcuts between browser sessions
  • Provide a personalized accessibility experience
  • Restore your settings if you reinstall the extension

Data Storage and Security

Local Storage Only: All extension data is stored locally in your browser using Chrome's secure storage API.

No Cloud Storage: No data is transmitted to external servers, databases, or cloud services.

No Tracking: We do not use cookies, analytics, or tracking technologies.

Extension Permissions

tradosClarity requests minimal permissions, each with a specific accessibility purpose:

  • activeTab: Required to enhance accessibility features on the current Trados Cloud page. This permission only activates when you click the extension icon or use a keyboard shortcut.
  • storage: Required to save your custom keyboard shortcut preferences locally in your browser.
  • Host permissions for *.trados.com: Required to run accessibility enhancements specifically on Trados Cloud domains.

Third-Party Services

tradosClarity does not integrate with any third-party services, analytics platforms, or external APIs. The extension operates entirely offline once installed.

Data Sharing

We do not share, sell, or distribute any data because we do not collect any data. Your keyboard shortcut preferences remain entirely private and local to your browser.

Children's Privacy

tradosClarity does not collect any information from users of any age. The extension is safe for users under 13 as no data collection occurs.

Open Source Transparency

tradosClarity is an open-source project. You can review the complete source code to verify our privacy practices at: https://github.com/paulfilkin/tradosClarity

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date and post the new policy. Since we do not collect contact information, we recommend checking this policy periodically for updates.

Compliance

  • Chrome Web Store Developer Program Policies
